# Decision Log

A personal decision-tracking web app focused on thinking accountability. Built with Next.js and Supabase.

## Core Features
- **Decision Capture**: Document your reasoning before you know the outcome.
- **Outcome Review**: Reflect on results after a cooling-off period.
- **Thinking Accountability**: Hindsight-bias protection through immutable reasoning logs.

## Tech Stack
- **Frontend**: Next.js (App Router)
- **Backend**: Supabase (Database + Auth)
- **Styling**: Vanilla CSS / Glassmorphism

## Infrastructure
- **Deployment**: Vercel
- **Database**: Supabase PostgreSQL + RLS

## Development
```bash
npm install
npm run dev
```
